Part 3 — Add starter PowerShell scripts

Create these scripts in:

C:\AI\voices\VoiceLab\scripts\

1. scan-voice-models.ps1

Purpose:
- Recursively scan C:\AI\voices
- Ignore VoiceLab output folders where appropriate to avoid scanning its own reports as models
- Detect likely model/tool folders
- Generate markdown and JSON inventory reports
- Print a terminal summary

2. create-character.ps1

Parameters:
- CharacterId
- DisplayName

Purpose:
- Create a new character folder under VoiceLab\characters
- Create the standard subfolders
- Create a starter voice_card.json
- Do not overwrite an existing character card unless -Force is passed
- Print success/failure messages

3. prepare-audio-for-voice-lab.ps1

Parameters:
- InputPath
- OutputPath
- SampleRate, default 44100
- Mono switch

Purpose:
- Use ffmpeg if available
- Convert input audio/video into clean WAV
- Optional mono conversion
- Print clear error if ffmpeg is missing
- Verify output exists

4. extract-ltx-audio.ps1

Parameters:
- VideoPath
- OutputWav

Purpose:
- Extract audio from an LTX-generated video into WAV using ffmpeg
- Preserve timing
- Verify output exists

5. remux-final-audio.ps1

Parameters:
- VideoPath
- AudioPath
- OutputVideoPath

Purpose:
- Replace the original video audio with final converted/mixed audio
- Use ffmpeg
- Copy video stream without re-encoding when possible
- Verify output exists

6. run-voice-lab-healthcheck.ps1

Purpose:
- Check that VoiceLab folders exist
- Check whether ffmpeg is available
- Check whether Python is available
- Print detected CUDA/NVIDIA info if available using nvidia-smi
- Print paths to detected model/tool folders from the inventory JSON if it exists
- Do not modify anything
